# Branddox web brand kit

## Put these files in your Next.js app
- `/public/favicon.ico`
- `/public/favicon-16x16.png`
- `/public/favicon-32x32.png`
- `/public/apple-touch-icon.png`
- `/public/android-chrome-192x192.png`
- `/public/android-chrome-512x512.png`
- `/public/site.webmanifest`

## Recommended structure
- `/public/brand/logo-primary.png`
- `/public/brand/logo-inverted.png`
- `/public/brand/logo-mono.png`
- `/public/brand/symbol.png`

## Recommended web colors
- Dark: #262833
- Mint: #00ED88
- White: #FFFFFF
- Light: #F3F4F6

## Recommended type pairing
- Headings: Sora or Poppins
- Body: Inter
